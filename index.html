运营信息


因 鄱阳路突发地陷，以下线路的运行受到不同程度的影响：

- 2号线（荷园--正门单向环回线）
- 3号线（正门--茶山广场班车）

为了保证乘客的人身安全以及车辆的安全运行，校巴方面决定对受影响的线路作出临时调整。

临时运行方案

1号线（西园--荷园）：

维持原线路不变。


2号线（荷园--正门单向环回线）：

正门 >> 湿地公园 >> 生命科学院区段停止运行。
开往荷园方向的车辆运行路线为正门 >> 艺术学院（食品学院）>> 钢铁研究所 >> 生命科学院 >> 荷园。
（原路返回）


3号线（正门--茶山广场班车）：

正门 -- 三角市 -- 钢铁研究所区段停止运行。
3号线全线车辆的运行路线变更为 正门 -- 艺术学院（食品学院）-- 钢铁研究所 -- 农学院 -- 资环学院 -- 嵩山小区（旧197总站）-- 校医院东门 -- 茶山广场。


N线（东区、启林区夜间快车）：

将视情况变更路线。乘车时请听从校巴师傅的安排。
